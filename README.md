# Balloon POP

# Description
- A simple browser game where players click on balloons to earn points before the timer runs out.

# Features
  - Floating balloons
  - Sound effects when a balloon is popped
  - Timer and score tracker
  - Start and end screen and buttons
 
  # Tools
  - HTML
  - CSS
  - JavaScript

  # Installation
  - No installation needed just click the demo link to play
